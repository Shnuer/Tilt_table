# Firmware for STM32F767ZI uC

## Pinout
- PE3 - Motor1 A
- PF7 - Motor1 B
- PF8 - Motor2 A
- PF9 - Motor2 B
- PD12  - Motor1 PWM
- PD13  - Motor2 PWM

- PA3  - Potentiometer Servo1
- PC0  - Potentiometer Servo2

- PC3  - Table ADC
- PD6  - Table Line
- PD5  - Table Line
- PD4  - Table Line
- PD3  - Table Line
